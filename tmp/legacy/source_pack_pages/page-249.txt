11. Component refining 
SystemName = Component Refining 
Material = Mount Vehicle Refine Essence 
Source = Component chests + events 
Effect = Changes component boost into random new boost 
Rules: 
Refining changes the component stat randomly. 
Refining cost increases every time you refine the same component. 
Better to refine Yellow or higher components. 
If you refine below Yellow, then later merge it, the refined stat can be lost/rerolled. 
Optimizer rule: 
if ComponentRarity < Gold: 
   RecommendRefine = False 
else: 
   RecommendRefine = True if player is targeting specific stat 
For damage-only optimizer, target stats are: 
Skill Damage
