3. Mount selection rule 
This is not a “priority chart.” This is a system rule from the guide: 
Use the highest unlocked mount. 
Doomsteed > Tech Hoverboard > Electric Scooter. 
Reason: 
Once the better mount is unlocked, use it even if the older mount has more stars. 
Older mounts still matter because their installed components give partial stats through Sync 
Rate. 
Optimizer logic: 
if DoomsteedUnlocked: 
   EquippedMount = Doomsteed 
elif TechHoverboardUnlocked: 
   EquippedMount = TechHoverboard 
else: 
   EquippedMount = ElectricScooter
